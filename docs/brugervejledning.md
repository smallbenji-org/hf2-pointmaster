# Bruger vejledning

## Oprettelse af bruger
1. Tryk signup i øverste højre hjørne
2. Indsat mail og kode
    - Der er ikke nogle krav til kodens længde, specialteng eller store og små bogstaver.
3. Det er nu muligt at oprette et nyt event

## Oprettelse af nyt event
1. Tryk "Tilføj event"
2. Giv event et navn

## Invitering af bruger til event
1. Tilgå members side
2. Tryk tilføj medlem og indtast brugerens mail
3. Brugeren vil nu få en invitation på deres side

## Oprettelse af poster
1. Tilgå poster siden
2. Tryk opret post
3. Indtast postens navn
4. Tryk opret

## Oprettelse af patuljer
1. Tilgå patruljer siden
2. Tryk opret patruljer
3. Indsat patruljens navn
4. Tryk opret
